Detailed Description

Advertising Parameters; for defaults see gGapDefaultAdvertisingParameters_d.

Field Documentation

bleAdvertisingType_t advertisingType

Advertising type. Default: connectable undirected.

gapAdvertisingChannelMapFlags_t channelMap

Bit mask indicating which of the three advertising channels will be used. Default: all three.

bleDeviceAddress_t directedAddress

Address of the target initiator; same as above.

bleAddressType_t directedAddressType

Address type of the target initiator; only used if advertisingType is equal to gAdvDirectedHighDutyCycle_c or gAdvDirectedLowDutyCycle_c.

gapAdvertisingFilterPolicyFlags_t filterPolicy

Indicates whether the connect and scan requests are filtered using the White List. Default: does not use White List (process all).

uint16_t maxInterval

Maximum desired advertising interval. Default: 1.28 s.

uint16_t minInterval

Minimum desired advertising interval. Default: 1.28 s.

bleAddressType_t ownAddressType

Indicates whether the advertising address will be the public address (BD_ADDR) or the random address (set by Gap_SetRandomAddress). Default: public address.
