uint32_t ft_control_get_electrodes_state ( struct ft_control *  control)

Get the state of all control electrodes.

Parameters
controlPointer to the control data.
Returns
This function returns a bit-mask value, where each bit represents one control electrode. Logic 1 in the returned value represents a touched electrode.
1  uint32_t touched_electrode = 0;
2  touched_electrode = ft_control_get_electrodes_state(&my_control);
3 printf("The electrode state is following: 0x%X in my control.", touched_electrode);
int32_t ft_control_get_touch_button ( const struct ft_control *  control,
uint32_t  index 
)

Get touched electrode.

Parameters
controlPointer to the control.
indexIndex of the first electrode to be probed. Use 0 during the first call. Use the last-returned index+1 to get the next touched electrode.
Returns
Index of the touched electrode, or FT_FAILURE when no electrode is touched.

Use this function to determine, which control elecrodes are currently touched. This is an example of getting the touched electrodes of control in the FT library:

1 int32_t last_touched_electrode = 0;
2 uint32_t electrode_count = ft_control_count_electrodes(&my_control);
3 last_touched_electrode = ft_control_get_touch_button(&my_control, last_touched_electrode);
4 while(last_touched_electrode != FT_FAILURE)
5 {
6  printf("The electrode %d in my control is touched", last_touched_electrode);
7  last_touched_electrode = ft_control_get_touch_button(&my_control, last_touched_electrode);
8 }
